准备工作

打开终端，装上本地连接器

在Web浏览器上安装一个插件

对应地址如下
对于 Chrome、Chromium 和 Vivaldi： Chrome Web 商店
对于 Firefox： Mozilla Addons
对于 Opera： Opera Addons

gnome插件

只记录了几个在用的或者觉得还不错的插件，还有一堆对我来说媒用的都没装，所有的gome桌面插件都可以在这里找到，也可以在这里删掉不想要了的插件
https://extensions.gnome.org/

bing-wallpaper-changer

每日自动更换必应壁纸，找了好久才找到
https://extensions.gnome.org/extension/1262/bing-wallpaper-changer/
本项目开源，至今一直在活跃中，github传送门
https://github.com/neffo/bing-wallpaper-gnome-extension

Battery Percentage

状态栏显示剩余电量显示
https://extensions.gnome.org/extension/818/battery-percentage/

Clipboard Indicator

在状态栏上显示一个按钮，点击可以显示最近的剪贴板
比如要从某个界面复制多次东西，可以连续复制几次，导入的时候从这里选择，方便快捷
https://extensions.gnome.org/extension/779/clipboard-indicator/

Drop down terminal

在tab上面的波浪号键上按一下就可以弹出一个terminal命令行界面，方便快捷
https://extensions.gnome.org/extension/442/drop-down-terminal/

Simple net speed

状态栏网速插件，点一下还可以切换显示方式
https://extensions.gnome.org/extension/1085/simple-net-speed/

User Themes

据说装了这个就可以用gnome shell使用各种主题了？没测试，但是先装上了
https://extensions.gnome.org/extension/19/user-themes/

SystemMonitor

一个可以在状态栏查看cpu负载情况的插件，然而没觉得知道这有什么用，卡顿照样卡
https://extensions.gnome.org/extension/9/systemmonitor/
